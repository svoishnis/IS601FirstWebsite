Unix/Linux Command Reference

File Commands

  • ls\t directory listing
  • ls-al\t formatted listing with hidden files
  • cd dir\t change directory to dir
  • cd\t change to home
  • pwd\t show current directory
  • mkdir dir\t create a directory dir
  • rm file\t delete file
  • rm -r dir\t delete directory dir
  • rm -f file\t force remove file
  • rm -rf dir\t force remove directory dir *
  • cp file1 file2\t copy file1 to file2
  • cp -r dir1 dir2\t copy dir1 to dir2; create dir2 if it doesn't exist
  • mv file1 file2\t rename or move file1 to file2 if file2 is an existing directory, moves file1 into directory file2
  • ln -s file link\t create symbolic link link to file
  • touch file\t create or update file
  • cat > file\t places standard input into file
  • more file\t output the contents of file
  • head file\t output the first 10 lines of file
  • tail file\t output the last 10 lines of file
  • tail -f file\t output the contents of file as it grows, starting with the last 10 lines

Process Management

  • ps\t display your currently active processes
  • top\t display all running processesv
  • -kill pid\t kill process id pid
  • killall proc\t kill all processes named proc *
  • bg\t lists stopped or background jobs; resume a stopped job in the background
  • fg\t brings the most recent job to foreground
  • fg n\t brings job n to the foreground

File Permissions

  • chmod octal file\t change the permissions of file to octal, which can be found separately for user, group, and world by adding:
      -
    • 4 read (r)
    • 2 write (w)
    • 1 execute (x)
    Examples:
  • chmod 777 – read, write, execute for all
  • chmod 755 – rwx for owner, rx for group and world For more options, see '**man chmod\t

SSH

  • ssh user@host\t connect to host as user
  • ssh -p port user@host\t connect to host on port port as user
  • ssh-copy-id user@host\t add your key to host for user to enable a keyed or passwordless login

Searching

  • grep pattern files\t search for pattern in files
  • grep -r pattern dir\t search recursively for pattern in dir
  • command | grep pattern\t search for pattern in the output of command
  • locate file\t find all instances of file

System Information

  • date\t show the current date and time
  • cal\t show this month's calendar
  • uptime\t show current uptime
  • w\t display who is online
  • whoami\t who you are logged in as
  • finger user\t display information about user
  • uname -a\t show kernel information
  • cat /proc/cpuinfo\t cpu information
  • cat /proc/meminfo\t memory information
  • man command\t show the manual for command
  • df\t show disk usage
  • du\t show directory space usage
  • free\t show memory and swap usagev
  • whereis app\t show possible locations of app which app – show which app will be run by default

Compression

  • tar cf file.tar files\t create a tar named file.tar containing files
  • tar xf file.tar\t extract the files from file.tar
  • tar czf file.tar.gz files\tcreate a tar with Gzip compressionv
  • tar xzf file.tar.gz\t extract a tar using Gzip
  • tar cjf file.tar.bz2\t create a tar with Bzip2 compression
  • tar xjf file.tar.bz2\t extract a tar using Bzip2
  • gzip file\t compresses file and renames it to file.gz
  • gzip -d file.gz\t decompresses file.gz back to file

Network

  • ping host\t ping host and output results
  • whois domain\t get whois information for domain
  • dig domain\t get DNS information for domain
  • dig -x host\t reverse lookup host
  • wget file\t download file
  • wget -c file\t continue a stopped download

Installation

    Install from source:
  • /configure\t
  • make\t
  • make install\t
  • dpkg -i pkg.deb\t install a package (Debian)
  • rpm -Uvh pkg.rpm\t install a package (RPM)

Shortcuts

  • Ctrl+C\t halts the current command
  • Ctrl+Z\t stops the current command, resume with fg in the foreground or bg in the background
  • Ctrl+D\t log out of current session, similar to exit
  • Ctrl+W\t erases one word in the current line
  • Ctrl+U\t erases the whole line
  • Ctrl+R\t type to bring up a recent command
  • !!\t repeats the last command
  • exit** log out of current session
* use with extreme caution.

VI Commands

  • vi filename edit the filename starting at line 1
  • vi -r filename recover filename that was being edited when system crased
  • :x quit vi, writing out modified file to file named in original invocation
  • :wq quit vi, writing out modified file to file named in original invocation
  • :q quit (or exit) vi
  • :q! quit vi even through latest changes have not been saved for this vi call
  • j or [or down-arrow] move cursor down one line
  • k [or up-arrow] move cursor up one line
  • h or [or left-arrow] move cursor left one character
  • l or [or right-arrow] move cursor right one character
  • 0 (zero) move cursor to the start of the current line
  • $ move cursor to the end of the current line
  • w move cursor to the beginning of the next word
  • b move cursor back to the beginning of the preceeding word
  • :0 of 1G move cursor to fist line in file
  • :n or nG move cursor to line n
  • :$ or G move cursor to las line in file
  • ^f move forward one screen
  • ^b move backward one screen
  • ^d move down (forward) one half screen
  • ^u move up (back) one half screen
  • ^l redraws the screen
  • ^r redraws the screen, removing deleted lines
  • u undo whatever you just did
  • i insert text before cursor, wait until hit
  • I insert test at beginning of current line, until hit
  • a append text after cursor, until hit
  • A append text to end of current line, until hit
  • o open and put text in a new line below current line, until hit
  • O open and put text in a new line above current line, until hit
  • r replace single character under cursor(no needed)
  • R replace characters, starting with the current position, until hit
  • cw change the current word with new text, starting with the character under cursor, until hitv
  • cNw change n words beginning with character under cursor, until hit
  • C Change (replace) the characters in the current line, until hit
  • cc change (replace) the entire current line, stopping when is hit
  • Nx or cNc change (replace) the next N lines, starting with the current line, stopping when hit
  • x delete single character under cursor
  • Nx delete N characters, starting with the character under cursor
  • dw delete the single word beginning with character under cursor
  • dNw delete N words beginning with character under cursor; d5w deletes 5 words
  • D delete the remainder of the line, starting with current cursor position
  • dd delete entire current line
  • Ndd or dNd delete N lines, beginning with the current line
  • yy copy (yank,cut) the current line into the buffer
  • Nyy or yNy copy (yank,cut) the next N lines, including the current lin, into the bufferv
  • p put (paste) the line(s) in the puffer into the text after the current line
  • /string search forward for occurrence of string in text
  • ?String search backward for occurence of string in text
  • n move to next occurrence of search string
  • N move to next occurrence of search string in opposite direction
  • :.= returns line number of current line at bottom of screen
  • := returns total number of lines at bottom of screen
  • ^g provides the current line number, along with the total umber of lines in the file at the bottom of the screen
  • :r filename read file named filename and insert after current line
  • :w write current contents to file named in original vi call
  • :w newfile write current contents to a new file named newfilev
  • :12,35w smallfile Writes the contents of the lines numbered 12 through 35 to a new ile nafmed smallfilev
  • :w! prevfile write current contents over a pre-existing file named prevfile
